# 柔術競賽

## 項目：寢技(Ne-Waza) 、對打(Fighting)、格鬥(Contact)

每單位每級報名人數以3人為限，每人限參加1個量級但參賽項目不限

## 男子組：

(1)-56 公斤級：56.00 公斤以下

(2)-62 公斤級：56.01 公斤~62.00 公斤

(3)-69 公斤級：62.01 公斤~69.00 公斤

(4)-77 公斤級：69.01 公斤~77.00 公斤

(5)-85 公斤級：77.01 公斤~85.00 公斤

(6)-94 公斤級：85.01 公斤~94.00 公斤

(7)+94 公斤級：94.01 公斤以上

## 女子組：

(1)-49 公斤級：49.00 公斤以下

(2)-55 公斤級：49.01 公斤~55.00 公斤

(3)-62 公斤級：55.01 公斤~62.00 公斤

(4)-70 公斤級：62.01 公斤~70.00 公斤

(5)+70 公斤級：70.01 公斤以上

# 比賽制度：

1. 寢技、對打、格鬥：採單淘汰制。
2. 對打每場3分鐘，寢技每場6分鐘，格鬥柔術預賽每場2分鐘+1分鐘黃金分鐘、準決賽及決賽3分鐘+(最多兩次2分鐘黃金分鐘) 黃金分鐘由場上裁判之多數判決勝負。

# 管理者：

管理員後台：

1. 負責管理及匯入隊伍及參賽者管理、設定場次及對戰名單。
2. 系統要檢查每人限參加一個組別、量級，且不可跨組別、跨量級參賽。

賽序裁判：負責管理賽序、計時、分數等相關事務。

# 賽程設計規格

## 籤表生成

- 每個項目（寢技、對打、格鬥）× 每個組別 × 每個量級，各自產生獨立的單淘汰籤表
- 系統依報名人數自動向上補齊至 2 的冪次（4、8、16…），多餘位置填入輪空（Bye）
- 輪空自動分配至籤表下方位置
- 管理員可手動調整籤表（交換位置、替換選手）

## 單淘汰晉級推進

- 裁判在計分介面輸入勝負結果後，系統自動將勝者帶入籤表下一回合
- 輪空的一方自動視為勝出，直接晉級下一回合
- 管理員可手動覆蓋任一場次結果（處理棄賽、取消或爭議）

## 場次衝突偵測

- 同一選手（因可跨項目參賽）若在同一時段被排入兩場比賽，系統自動標示衝突警告
- 衝突警告為視覺提示（標示橘色/紅色），不阻擋賽程繼續
- 賽序裁判可手動調整場次順序以消除衝突

## 場地配置

- 本賽事採單一場地，所有場次依序排列於同一時間線上

# 計分賽序流程

## 各項目計時規則

| 項目             | 階段         | 比賽時間 | 延長                           |
| ---------------- | ------------ | -------- | ------------------------------ |
| 寢技（Ne-Waza）  | 全程         | 6 分鐘   | 無                             |
| 對打（Fighting） | 全程         | 3 分鐘   | 無                             |
| 格鬥（Contact）  | 預賽         | 2 分鐘   | 無                             |
| 格鬥（Contact）  | 準決賽、決賽 | 3 分鐘   | 最多 2 次黃金分鐘（各 2 分鐘） |

## 比賽流程

1. 賽序裁判從籤表選取當前場次，確認紅方 / 藍方選手
2. 賽序裁判啟動計時
3. 記錄裁判視比賽狀況在計分介面即時記錄得分（對打項目可記錄一本）
4. 時間到（或對打達成一本條件）→ 裁判判決確認勝者（點選紅方 / 藍方）
5. 系統記錄結果，自動將勝者推進籤表下一回合

## 計分介面規格

### 寢技（Ne-Waza）

```
┌─────────────────────────────────────────────────┐
│  寢技 Ne-Waza · 男子組 · -62 公斤級          │
│                [計時器]  06:00                   │
├───────────────────────┬─────────────────────────┤
│  紅方                 │   藍方                   │
│  [隊名]               │   [隊名]                 │
│  [姓名]               │   [姓名]                 │
│  得分：0  優勢：0      │   得分：0  優勢：0        │
│  [+2] [+3] [+4] [+A] [+P]    │   [+2] [+3] [+4] [+A] [+P]        │
│  [-2] [-3] [-4] [-A] [-P]    │   [-2] [-3] [-4] [-A] [-P]        │
│  [降伏勝] [DQ] [傷停]         │   [降伏勝] [DQ] [傷停]           │
├───────────────────────┴─────────────────────────┤
│  裁判判決：  [紅方勝]  [藍方勝]                  │
└─────────────────────────────────────────────────┘
```

**得分按鈕對應技術：**

| 按鈕     | 分值   | 對應技術                             |
| -------- | ------ | ------------------------------------ |
| [2分]    | 2      | 翻轉、摔倒、膝蓋壓制                 |
| [3分]    | 3      | 過腿                                 |
| [4分]    | 4      | 騎態、背位                           |
| [優勢]   | 優勢+1 | 裁判給予優勢，另行顯示不計入正式得分 |
| [降伏勝] | 99     | 降伏，場次立即結束，該方勝出         |

**取消機制：**

- **[-警告]**：該方警告數 -1（最低為 0）；若取消前一次警告已自動觸發對方優勢，系統同步扣回對方該優勢，並提示確認
- **+警告**：每點一次該方警告數 +1，依累計次數自動對對方補優勢或判負（見下方規則）
- **+傷停**：該方選手傷停，觸發傷停計時；詳見傷停規格
- **DQ**：該方選手嚴重犯規，點選後提示賽序裁判確認，確認後該方判負，對方自動晉級
- 勝負完全以裁判判決確認（降伏勝或 DQ 除外，自動結束）

#### 寢技警告累計規則

| 警告次數 | 對方自動得分 | 系統行為                             |
| -------- | ------------ | ------------------------------------ |
| 1 次     | 無           | 單純記錄                             |
| 2 次     | 對方 +1 優勢 | 自動加優勢並提示                     |
| 3 次     | 對方 +2 分   | 對方自動加2分並提示                  |
| 4 次     | DQ、對方勝   | 提示賽序裁判確認，確認後立即結束場次 |

> 優勢獨立顯示於計分板，不計入正式得分；第 2、3 次警告的優勢由系統自動執行，不需賽序裁判手動點選。

裁判操作畫面

```
┌─────────────────────────────────────────────────┐
│  對打 Fighting · 男子組 · -62 公斤級        │

├───────────────────────┬─────────────────────────┤
│  紅方                 │   藍方                   │
│  [隊名]               │   [隊名]                 │
│  [姓名]               │   [姓名]                 │
│  得分：0              │   得分：0                 │
│  [+1] [+2] [+3] [+4] [+A] [+P]       │   [+1] [+2] [+3] [+4] [+A] [+P]        │
│  [-1] [-2] [-3] [-4] [-A] [-P]      │   [-1] [-2] [-3] [-4] [-A] [-P]         │
│  一本：               │   一本：                  │
│  [打×0] [摔×0] [寢×0] │   [打×0] [摔×0] [寢×0]   │
│  [+傷停]  [DQ]        │   [+傷停]  [DQ]           │

│  裁判判決：  [紅方勝]  [藍方勝]                  │
└─────────────────────────────────────────────────┘
```

- 加分 / 減分：統一計分，各 1～3 分，無分區塊
- **一本區塊**：分「打」「摔」「寢」三個按鈕，各自獨立累計次數，每點一次 +1（顯示 `打×1`、`摔×2`…）
- **+傷停**：該方選手傷停，觸發傷停計時；詳見傷停規格
- **DQ**：該方選手嚴重犯規，點選後提示賽序裁判確認，確認後該方判負，對方自動晉級
- 得分與一本次數即時顯示
- 最終勝負以裁判判決按鈕確認為準

- 準決賽 / 決賽時間到無結果時，進入黃金分鐘流程（詳見下方）

## DQ（嚴重犯規）規格

- **觸發**：賽序裁判在紅方或藍方區域點選「DQ」按鈕
- **行為**：
  1. 系統顯示警示彈窗，請賽序裁判確認 DQ
  2. 確認後：該方選手判負，對方自動晉級，場次立即結束
- **記錄**： DQ 原因可由賽序裁判輸入備註，記錄於場次資料中

## 計時器設定規格

計時器分兩個使用情境：**場次開始前設定**，以及**比賽暫停中調整**。

```
場次開始前（選擇比賽時間）：
┌─────────────────────────────────────────────────┐
│               設定比賽時間                       │
│                                                  │
│  快速選擇：[2分] [3分] [5分] [6分]               │
│                                                  │
│              [確認開始比賽]                      │
└─────────────────────────────────────────────────┘

比賽暫停中（臨時調整剩餘時間）：
┌─────────────────────────────────────────────────┐
│  ⏸ 暫停中       剩餘時間：02:30                  │
│                                                  │
│  調整時間：  分 [ ▲ 02 ▼ ]  秒 [ ▲ 30 ▼ ]      │
│                                                  │
│         [繼續比賽]   [儲存並繼續]                │
└─────────────────────────────────────────────────┘
```

- **快速選擇**（開始前）：點選預設按鈕（2分 / 3分 / 5分 / 6分）帶入對應時間；依賽制自動帶入預設值，賽序裁判可覆蓋
- **暫停中微調**：計時器暫停時，賽序裁判可透過 ▲ / ▼ 調整分鐘（0～99）與秒數（0～59），用於臨時加時等狀況
  - `[繼續比賽]`：不修改時間，直接恢復倒數
  - `[儲存並繼續]`：套用調整後的時間，再恢復倒數
- 時間調整操作記錄於場次記錄中（含調整前後時間）

- **觸發**：賽序裁判按下「+傷停」按鈕
- **行為**：
  1. 比賽計時器立即暫停
  2. 畫面切換顯示傷停倒數計時器，從 **2:00** 開始倒數
  3. 畫面顯示「傷停處理中」的明顯提示
- **傷停結束**：
  - 傷停計時倒數至 0：自動提示賽序裁判復賽
  - 賽序裁判手動提前結束傷停：按下「繼續比賽」，恢復比賽計時器
- **記錄**：傷停次數與時間記錄於本場比賽資料中
- **無限制次數**：本規格不限制單場傷停次數，由賽序裁判判斷

傷停中畫面：
┌─────────────────────────────────────────────────┐
│               ⚠ 傷停處理中                       │
│                                                  │
│           傷停倒數  01:45                        │
│                                                  │

## 黃金分鐘規則（格鬥準決賽 / 決賽）

- 正規時間結束後若無明確結果，進入黃金分鐘延長
- 每次延長 2 分鐘，最多進行 2 次
- 黃金分鐘結束由場上裁判多數決定勝負（需3位裁判投票，取多數）
- 系統顯示第 N 次黃金分鐘標示，並提供裁判投票按鈕（紅方 / 藍方）

## 賽序裁判操作流程

1. **建立籤表**：管理員匯入選手後，由系統自動生成各項目籤表
2. **調整籤表**：管理員確認或手動調整籤表順序
3. **衝突檢查**：系統標示跨項目選手的時間衝突，賽序裁判手動調整場次順序
4. **開始場次**：賽序裁判點選場次，啟動計時
5. **記錄結果**：記錄裁判輸入得分，裁判判決確認勝負
6. **推進籤表**：系統自動將勝者帶入下一回合

# 傷停規則

運動項目：寢技、對打(2分鐘)、格鬥(3分鐘)

1. 傷停的 icon：`faBriefcaseMedical`，修改成不要用 icon，改成用 CSS 來繪製：底為紅色、字為「+」、字體白色粗體、圓角，大小與傷停按鈕相同，可用 Card 方式呈現。裁判端和觀眾端都要修改。

2. 傷停時間上限（每位選手每場比賽）：
   - 寢技：**120 秒**
   - 對打（2 分鐘比賽）：**120 秒**
   - 格鬥（3 分鐘比賽）：**180 秒**
   - 傷停時間依比賽類型動態設定，不可寫死。

3. 傷停規則：
   - 傷停時間是**累積制**，每位選手每場比賽的傷停時間為累計計算。
   - 啟動傷停後，選手的比賽時間暫停，傷停倒數開始計時。
   - 選手或裁判結束傷停時，傷停倒數**暫停**，但傷停倒數顯示**不關閉**，讓選手與觀眾知道下次傷停還剩多少時間。
   - 選手再次啟動傷停時，從上次剩餘時間繼續倒數（不重設），直到傷停時間全數用完。
   - 傷停時間歸零後，不可再啟動傷停。
   - 傷停結束（繼續比賽）後，選手的比賽時間繼續倒數。

# 裁判計分版修正

1. 原來紅、藍方底部操作列的「取消（取消上一筆）」按鈕移除。不需要此功能。
2. 紅方計分區底色改為淡粉紅色，藍方計分區底色改為淡粉藍色。

# 觀眾端修正

1. 紅方計分區加框線用淡粉紅色，藍方計分區加框線用淡粉藍色。並加上陰影。
2. 紅方和藍方的得分（大分數）區塊上方，各加上「得分」標籤文字。
3. 警告及優勢的位置交換（原本由左至右為：優勢 → 警告，改為：警告 → 優勢）。
4. 警告的呈現方式改為黃牌：以 2×2 排列四個方框，當選手有警告時，對應數量的框變為黃色，其餘為灰色框。
   - 警告上限為 **4 個**，達到上限後警告按鈕停用，不可再加。

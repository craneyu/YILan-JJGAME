# 賽會競賽項目不同處

- 1.保留現有系統賽制與操作流程。目前已開發的屬於宜蘭縣運動會專用。運動規則和相關規定將繼續適用於宜蘭縣運動會，並且不會受到新的系統規格的影響。
- 2.建立新的系統規格。這些新的系統規格將除了國小組有特殊限制，其它與現有的宜蘭縣運動會專用系統設計是一樣的，但不要影響原系統的運作。
- 3.新的運動賽會由宜蘭縣柔術錦標賽
- 4.新的運動賽會項目：雙人傳統演武、雙人創意演武、寢技、對打、格鬥、柔術技

# 賽會競賽項目不同處

## 演武

### 雙人傳統演武

分組一：男子組、女子組、混合組
分組二：國小低年級組、國小中年級組、國小高年級組、國高中組、公開組

動作規定：
國小低年級組：A1、B1。 一次2個動作演練完，裁判計分後，換下一組動作演練。
國小中年級組：A1、A2、B1、B2。 一次4個動作演練完，裁判計分後，換下一組動作演練。

國小高年級組：A1～A3、B1～B3、C1～C3。 與國高中組、公開組相同，A系列打完裁判計分後，換下一組動作演練。再演練B系列，待裁判對B系列計分後，最後演練C系列。與原系來的演練流程一致。

國小組：不需要VR裁判，裁判以動作完成度為評分依據。

國高中組及公開組：依照原系統規格

### 雙人創意演武

分組如上

比賽時間：
國小組：超時或未達時間者，不扣分。
國高中組及公開組：依照原系統規格，不得少於1分鐘，不得多於2分鐘，超時或未達時間者，扣1分。

### 柔術技(獨立全新項目)

分組一：男子組、女子組
分組二：國小低年級組、國小中年級組、國小高年級組、國高中組、公開組
5位裁判評分，10分制，小數1位。去頭尾分後加總分數。

### 寢技

分組一：男子組、女子組
分組二：國小組、國高中組、公開組

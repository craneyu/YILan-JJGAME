# 宜蘭縣柔術錦標賽 — 系統需求規格

> 版本：v2.0 ｜ 整理日期：2026-04-10

## 1. 總覽

### 1.1 基本原則

1. **保留現有系統**：宜蘭縣運動會的賽制與操作流程完全不受影響
2. **新增賽會類型**：在同一平台中新增「宜蘭縣柔術錦標賽」作為獨立賽會
3. **共用核心架構**：認證、Socket.IO、資料庫等基礎設施共用，競賽規則各自獨立

### 1.2 賽會競賽項目總覽

| #   | 競賽項目     | 現有系統對應              | 差異程度    | 說明                                       |
| --- | ------------ | ------------------------- | ----------- | ------------------------------------------ |
| 1   | 雙人傳統演武 | Kata（傳統演武）          | 🟡 中度差異 | 國小組有動作數量限制、輪次減少、無 VR 裁判 |
| 2   | 雙人創意演武 | Creative Embu（創意演武） | 🟢 小差異   | 國小組超時/不足時間不扣分                  |
| 3   | 柔術技       | ❌ 無（全新項目）         | 🔴 全新開發 | 10 分制、小數 1 位、5 裁判去頭尾加總       |
| 4   | 寢技         | Ne-Waza（寢技）           | 🟢 小差異   | 新增國小組分組                             |
| 5   | 對打         | Fighting（對打）          | ⚪ 無差異   | 完全沿用現有規則                           |
| 6   | 格鬥         | Fighting/格鬥             | ⚪ 無差異   | 完全沿用現有規則                           |

## 2. 跨項目共通變更：分組制度

### 2.1 現有系統分組

現有系統僅有**一級分組**（性別分組）：

- 男子組（MALE）
- 女子組（FEMALE）
- 混合組（MIXED）

### 2.2 錦標賽新增二級分組（年齡/級別）

錦標賽在性別分組之外，新增**二級分組**（年齡/級別分組）：

| 二級分組     | 代碼 | 適用項目                     |
| ------------ | ---- | ---------------------------- |
| 國小低年級組 | EL   | 傳統演武、創意演武、柔術技   |
| 國小中年級組 | EM   | 傳統演武、創意演武、柔術技   |
| 國小高年級組 | EH   | 傳統演武、創意演武、柔術技   |
| 國高中組     | JH   | 傳統演武、創意演武、柔術技   |
| 公開組       | OPEN | 傳統演武、創意演武、柔術技   |
| 國小組       | ELEM | 寢技（不再細分低/中/高年級） |

> **排名計算**：同一級分組 × 同二級分組內的隊伍互相排名
> 例如：男子組 × 國小高年級組 為一個獨立的排名群組

### 2.3 各項目適用的分組組合

| 競賽項目     | 一級分組（性別） | 二級分組（年齡/級別）                            |

| 雙人創意演武 | 男子、女子、混合 | 國小低年級、國小中年級、國小高年級、國高中、公開 |

## 3. 項目一：雙人傳統演武（差異規格）

> 國高中組及公開組完全沿用現有系統規格，以下僅描述**國小組的差異**。

### 3.1 動作數量限制

| 二級分組     | 可用動作               | 系列結構                                                                    |
| ------------ | ---------------------- | --------------------------------------------------------------------------- |
| 國小低年級組 | A1、B1                 | A 系列 1 動作 + B 系列 1 動作，共 2 動作，**2 輪**（R1=A、R2=B）            |
| 國小中年級組 | A1、A2、B1、B2         | A 系列 2 動作 + B 系列 2 動作，共 4 動作，**2 輪**（R1=A、R2=B）            |
| 國小高年級組 | A1～A3、B1～B3、C1～C3 | A/B/C 各 3 動作，共 9 動作，**3 輪**（R1=A、R2=B、R3=C）                    |
| 國高中組     | A1～A4、B1～B4、C1～C4 | 沿用現有（A/B/C 各 4 動作，分男子組與女子組、混合組，不同動作數），**3 輪** |
| 公開組       | A1～A4、B1～B4、C1～C4 | 沿用現有（A/B/C 各 4 動作，分男子組與女子組、混合組，不同動作數），**3 輪** |

### 3.2 國小低/中年級：評分流程

評分流程：**與現有系統完全相同**（逐動作評分），僅動作數量不同：

- 每個動作的裁判介面、評分操作與現有系統一模一樣
- 計分規則不變：每項仍是 5 位裁判去頭尾取中間 3 位加總

**國小低年級（2 動作、2 輪）**：

```
R1（A 系列）：演練 A1 → 開放評分 → 裁判評 P1～P4 送出 → 換組（所有隊伍輪完）
R2（B 系列）：演練 B1 → 開放評分 → 裁判評 P1～P4 送出 → 換組（所有隊伍輪完）
```

**國小中年級（4 動作、2 輪）**：

```
R1（A 系列）：演練 A1 → 開放評分 → 裁判評 P1～P4 送出
             → 演練 A2 → 開放評分 → 裁判評 P1～P4 送出 → 換組（所有隊伍輪完）
R2（B 系列）：演練 B1 → 開放評分 → 裁判評 P1～P4 送出
             → 演練 B2 → 開放評分 → 裁判評 P1～P4 送出 → 換組（所有隊伍輪完）
```

### 3.3 國小組：無 VR 裁判

- 國小所有年級組（低年級、中年級、高年級）均**不需要 VR 裁判**
- 不進行摔技/地板技多樣性評分
- 不進行錯誤攻擊標記
- VR 裁判介面對國小組隊伍**不顯示**或**停用**
- 排名計算中不包含 VR 分數

### 3.4 國小高年級組流程

- 動作數量：A1～A3、B1～B3、C1～C3（每系列 3 動作）
- 評分流程：**與現有系統完全相同**（逐動作評分）
  - 演練 A1 → 開放評分 → 裁判評 P1～P4 送出 → 演練 A2 → 開放評分 → 裁判評分送出 → 演練 A3 → ... 依序完成
  - A 系列完成 → B 系列 → C 系列，輪次結構不變
- 唯一差異：動作數量較少（每系列 3 動作）+ **無 VR 裁判**
  - 系列完成後不需等待 VR 裁判送出多樣性評分
  - 賽序裁判可直接進行「換組」，無需等待 VR 評分

### 3.5 計分規則（所有組別通用）

與現有系統完全相同：

- 每個評分項目：5 位裁判評分，去最高、去最低，中間 3 位加總（最高 9 分）
- A/B 系列每動作：4 項 × 9 = 36 分
- C 系列每動作：5 項 × 9 = 45 分（國小高年級以上適用）

## 5. 項目三：柔術技（全新項目）

### 5.1 項目性質

- 這是一個**全新的競賽項目**，現有系統中無對應功能
- 為個人或雙人的技術展演評分

### 5.2 分組

| 一級分組       | 二級分組                                         |
| -------------- | ------------------------------------------------ |
| 男子組、女子組 | 國小低年級、國小中年級、國小高年級、國高中、公開 |

> 注意：柔術技**無混合組**

### 5.3 評分規則

| 項目     | 規格                                        |
| -------- | ------------------------------------------- |
| 裁判人數 | 5 位                                        |
| 分數範圍 | 0.0 ~ 10.0（10 分制）                       |
| 分數精度 | 小數 1 位（0.1 為最小單位）                 |
| 計分方式 | 去最高分、去最低分，加總中間 3 位裁判的分數 |
| 滿分     | 10.0 × 3 = **30.0 分**                      |
| 最低分   | 0.0 × 3 = **0.0 分**                        |

### 5.4 評分流程

一個場次同時最多 1～5 人上場展演，全部展演完畢後逐人評分：

```
1～5 人同時上場展演 → 全部展演完畢
→ 賽序裁判開放第 1 位選手評分 → 5 位裁判各自給分（0.0~10.0）→ 送出
→ 賽序裁判開放第 2 位選手評分 → 5 位裁判各自給分 → 送出
→ ... 依序評完所有選手
→ 換下一場次
```

### 5.5 裁判介面需求

- **計分裁判**：數字鍵盤輸入，範圍 0.0 ~ 10.0，步進 0.1
  - 選分完成後按「確認送出」
- **賽序裁判**：控制「開放評分」→「換人」流程（無系列/動作/輪次概念）
- **觀眾顯示**：顯示選手姓名、5 位裁判分數、去頭尾後的加總分、排名

### 5.6 無 VR 裁判

柔術技項目**不需要 VR 裁判**，無多樣性評分、無錯誤攻擊標記。

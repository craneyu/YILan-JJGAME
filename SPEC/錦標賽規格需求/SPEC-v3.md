## 1. 總覽

### 1.1 基本原則

1. **保留現有系統**：宜蘭縣運動會的賽制與操作流程完全不受影響
2. **新增賽會類型**：在同一平台中新增「宜蘭縣柔術錦標賽」作為獨立賽會
3. **共用核心架構**：認證、Socket.IO、資料庫等基礎設施共用,競賽規則各自獨立

### 1.2 賽會競賽項目總覽

| #   | 競賽項目     | 現有系統對應     | 差異程度    | 說明                                       |

| 1   | 雙人傳統演武 | Duo（傳統演武）  | 🟡 中度差異 | 國小組有動作數量限制、輪次減少、無 VR 裁判 |
| 2   | 雙人創意演武 | Show（創意演武） | 🟢 小差異   | 國小組超時/不足時間不扣分                  |
| 3   | 寢技         | Ne-Waza（寢技）  | 🟢 小差異   | 新增國小組分組                             |
| 4   | 對打         | Fighting（對打） | ⚪ 無差異   | 完全沿用現有規則                           |
| 5   | 格鬥         | Contact/格鬥     | ⚪ 無差異   | 完全沿用現有規則                           |

## 3. 項目一：雙人傳統演武（差異規格）

> 青少年國中組、青少年高中組及公開組完全沿用現有系統規格，以下僅描述**國小組的差異**。

### 3.1 動作數量限制

| 二級分組     | 可用動作               | 系列結構                                                                    |
| ------------ | ---------------------- | --------------------------------------------------------------------------- |
| 國小低年級組 | A1、B1                 | A 系列 1 動作 + B 系列 1 動作，共 2 動作，**1 輪**（同一組依序演練 A1 → B1 完畢後才換下一組） |
| 國小中年級組 | A1、A2、B1、B2         | A 系列 2 動作 + B 系列 2 動作，共 4 動作，**1 輪**（同一組依序演練 A1 → A2 → B1 → B2 完畢後才換下一組） |
| 國小高年級組 | A1～A3、B1～B3、C1～C3 | A/B/C 各 3 動作，共 9 動作，**3 輪**（R1=A、R2=B、R3=C）                    |
| 青少年國中組（男子組）         | A1～A4、B1～B4、C1～C4 | A/B/C 各 4 動作，共 12 動作，**3 輪**（沿用現有）         |
| 青少年國中組（女子組、混合組） | A1～A3、B1～B3、C1～C3 | A/B/C 各 3 動作，共 9 動作，**3 輪**（沿用現有）          |
| 青少年高中組（男子組）         | A1～A4、B1～B4、C1～C4 | A/B/C 各 4 動作，共 12 動作，**3 輪**（沿用現有）         |
| 青少年高中組（女子組、混合組） | A1～A3、B1～B3、C1～C3 | A/B/C 各 3 動作，共 9 動作，**3 輪**（沿用現有）          |
| 公開組（男子組）               | A1～A4、B1～B4、C1～C4 | A/B/C 各 4 動作，共 12 動作，**3 輪**（沿用現有）         |
| 公開組（女子組、混合組）       | A1～A3、B1～B3、C1～C3 | A/B/C 各 3 動作，共 9 動作，**3 輪**（沿用現有）          |

### 3.2 國小低/中年級：評分流程

評分流程：**同一組上台後依序完成所有動作評分，才換下一組上台**（與現有系統 R1=A 全部隊伍→R2=B 全部隊伍 的輪次結構不同）。每個動作的裁判介面、評分操作與現有系統一模一樣，計分規則不變：每項仍是 5 位裁判去頭尾取中間 3 位加總。

**國小低年級（2 動作、1 輪）**：

```
隊伍 N 上台：
  演練 A1 → 開放評分 → 裁判評 P1～P4 送出
  → 演練 B1 → 開放評分 → 裁判評 P1～P4 送出
→ 換組（換隊伍 N+1 上台，重複上述流程）
```

**國小中年級（4 動作、1 輪）**：

```
隊伍 N 上台：
  演練 A1 → 開放評分 → 裁判評 P1～P4 送出
  → 演練 A2 → 開放評分 → 裁判評 P1～P4 送出
  → 演練 B1 → 開放評分 → 裁判評 P1～P4 送出
  → 演練 B2 → 開放評分 → 裁判評 P1～P4 送出
→ 換組（換隊伍 N+1 上台，重複上述流程）
```

> **流程差異重點**：國小低/中年級的「換組」發生在同一隊伍完整演完 A、B 全部動作之後，而非演完一個動作就換到下一隊。等同把現有系統的「R1=全部隊伍 A → R2=全部隊伍 B」改成「每隊一次演完 A+B 再換隊」。

### 3.3 國小組：無 VR 裁判

- 國小所有年級組（低年級、中年級、高年級）均**不需要 VR 裁判**
- 不進行摔技/地板技多樣性評分
- 不進行錯誤攻擊標記
- VR 裁判介面對國小組隊伍**不顯示**或**停用**
- 排名計算中不包含 VR 分數

### 3.4 國小高年級組流程

- 動作數量：A1～A3、B1～B3、C1～C3（每系列 3 動作）
- 評分流程：**與現有系統完全相同**（逐動作評分）
  - 演練 A1 → 開放評分 → 裁判評 P1～P4 送出 → 演練 A2 → 開放評分 → 裁判評分送出 → 演練 A3 → ... 依序完成
  - A 系列完成 → B 系列 → C 系列，輪次結構不變
- 唯一差異：動作數量較少（每系列 3 動作）+ **無 VR 裁判**
  - 系列完成後不需等待 VR 裁判送出多樣性評分
  - 賽序裁判可直接進行「換組」，無需等待 VR 評分

### 3.5 計分規則（所有組別通用）

與現有系統完全相同：

- 每個評分項目：5 位裁判評分，去最高、去最低，中間 3 位加總（最高 9 分）
- A/B 系列每動作：4 項 × 9 = 36 分
- C 系列每動作：5 項 × 9 = 45 分（國小高年級以上適用）

### 3.5.1 排序與賽程推進規則（錦標賽）

**排序優先順序**：`tier（年齡別）→ category（性別組）→ order（隊伍場次）`

tier 固定順序為年齡升冪：

```
EL（國小低年級）→ EM（國小中年級）→ EH（國小高年級）
→ JH（青少年國中組）→ SH（青少年高中組）→ OPEN（公開組）
```

ELEM tier 寢技專用，不參與演武排序。Category 順序由 admin 設定（預設 `女子→男子→混合`，可於賽會管理介面調整）。

**nextGroup（換組）推進規則**：

- **EL / EM tier（單輪連續演練）**：每隊上台演完 A+B 全部動作為一個推進單元。同 (tier, category) 內逐隊推進；該 (tier, category) 完成 → 同 tier 下一非空 category；該 tier 完成 → 下一非空 tier。
- **EH / JH / SH / OPEN tier（多輪）**：以 `(tier, category, round)` 為推進單元。同 (tier, category, round) 內逐隊；同 tier 同 round 跑完所有 category → 同 tier round+1；同 tier round=3 跑完 → 下一非空 tier round=1。VR 評分仍為 round 邊界閘門。
- **空 (tier, category) 群組**：admin 設定 categoryOrder 含某 category 但該 tier 無隊伍報名時，nextGroup 自動跳過該空群組，無需手動處理。
- **單一隊伍 (tier, category) 群組**：完成所有動作（含 VR 條件）後，「換組」按鈕 enabled，nextGroup 推進到下一非空 (tier, category)。Sports-day（所有 tier=null）行為等同單一虛擬 tier，與本變更前完全一致。

**運動會回歸保證**：sports-day 賽會所有隊 tier=null，tier 主鍵全部同 rank → 退化為 category 主排序 → 與既有行為完全一致。

### 3.6 單一隊伍組別的輪次省略規則（適用所有採輪次結構的組別）

若某個「一級分組 × 二級分組」群組內**只有一隊參賽**，則該群組不進行輪次輪轉，由該隊一次性連續演完所有動作評分後即完成。

**適用範圍**：

- 國小高年級組（原本 R1=A → R2=B → R3=C 三輪輪轉）
- 青少年國中組（原本 R1/R2/R3 三輪輪轉）
- 青少年高中組（原本 R1/R2/R3 三輪輪轉）
- 公開組（原本 R1/R2/R3 三輪輪轉）

**單隊情境下的流程範例（以青少年高中組男子組為例，A/B/C 各 4 動作）**：

```
單一隊伍上台：
  演練 A1 → 評分 → A2 → 評分 → A3 → 評分 → A4 → 評分
  → 演練 B1 → 評分 → B2 → 評分 → B3 → 評分 → B4 → 評分
  → 演練 C1 → 評分 → C2 → 評分 → C3 → 評分 → C4 → 評分
→ 該組別完賽（無「換組」動作）
```

> **判定時機**：以該「一級分組 × 二級分組」群組於賽前報名後的隊伍數量為準。國小低/中年級因本身就是單輪連續演練（見 §3.2），不受此規則影響。

**賽序裁判介面行為（單隊組別）**：

- **「換組」按鈕停用**（按了無下一組可換，避免誤觸）
- **無需手動換輪**：A 系列最後一動作（如 A4）評分完成後，賽序裁判直接「開放下一動作評分」即進入 B1，再進入 B2... 系統依固定序列 A→B→C 自動推進，無「換輪」動作
- **VR 評分時機（青少年國中/青少年高中/公開組適用）**：A 系列全部評完 → VR 裁判送出 A 系列多樣性分數 → 再開放 B1；B 系列同理。VR 評分仍維持「系列結束後」的時機，不因單隊而省略

**觀眾顯示行為（單隊組別）**：

- 該群組只有一隊時，**隱藏 G（Group）標籤**——因為 G 必然永遠是 G1，顯示無意義
- 該群組只有一隊時，**隱藏 R（Round）標籤**——因為無輪次輪轉概念，標 R1/R2/R3 反而誤導
- 其餘群組（多隊參賽）的觀眾顯示維持現有 R / G 標籤行為不變

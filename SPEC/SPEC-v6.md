## 計時器設定規格

對打計時器分兩個使用情境：**場次開始前設定**，以及**比賽暫停中調整**。

```
場次開始前（選擇比賽時間）：
┌─────────────────────────────────────────────────┐
│               設定比賽時間                       │
│                                                  │
│  快速選擇：[2分] [3分]                            │
│                                                  │
│              [確認開始比賽]                      │
└─────────────────────────────────────────────────┘

比賽暫停中（臨時調整剩餘時間）：
┌──────────────────────────────────────────────────────────────────┐
│  ⏸ 暫停中       剩餘時間：02:30                                   │
│                                                                   │
│  快速調整：[▲1秒] [▼1秒] [▲10秒] [▼10秒] [▲1分] [▼1分]          │
│                                                                   │
│              [繼續比賽]   [儲存並繼續]                             │
└──────────────────────────────────────────────────────────────────┘
```

- **快速選擇**（開始前）：點選預設按鈕（2分 / 3分）帶入對應時間；依賽制自動帶入預設值，賽序裁判可覆蓋
- **暫停中微調**：計時器暫停時，賽序裁判可透過快速按鈕調整時間（支援秒級與分級調整），用於臨時加時等狀況
  - `[繼續比賽]`：不修改時間，直接恢復倒數
  - `[儲存並繼續]`：套用調整後的時間，再恢復倒數
- 時間調整操作記錄於場次記錄中（含調整前後時間）
- OSAE KOMI(壓制) 時間計數器預設為 15 秒.再次點選則裁判計分板重置為 15 秒，觀眾端畫面則顯示「OSAE KOMI 15秒」的提示訊息，並開始倒數；倒數結束後或裁判重置，自動關閉計數畫面
- MEDICAL(傷停) 時間計數器預設為 120 秒
- 加分 / 減分：每個 PART 區塊分別有 +2、+3、-2、-3 四個按鈕，分別對應加減 2 分或 3 分；ALL PARTS 的 +1/-1 按鈕則直接對**總分**加減 1 分（不計入任何 PART）
- 輕度犯規將被以「SHIDO(指導)」處罰，並給予對手 1 個 「WAZA-ARI」
- 中度犯規將以「CHUI(注意)」處罰，並給予對手3個「WAZA-ARI」，畫面(裁判計分板及觀眾端)要有「CHUI」的顯示，要有「-CHUI」，累積6個「WAZA-ARI」則視為DQ，對方勝
- 重度犯規將將被判 DQ，對方勝
- PART 區塊：分「PART 1」「PART 2」「PART 3」三個按鈕區，分別對應比賽的三個階段，每區塊（顯示 `+2`、`+3`及 `-2`、`-3`）；ALL PARTS 按鈕則直接對總分部份進行加減分 ;
- 成績上方紀錄PART依 PART1 PART2 PART3 順序顯示
- 每次點擊 `+2` 或 `+3`：該 PART 得分增加對應分數，同時 **IPPON 計數 +1**；點擊 `-2` 或 `-3`：得分減少，IPPON 計數 -1（最低為 0）。同一個 PART 可累積多次 IPPON。
- 三個 PART 的 IPPON 次數**各自都大於 0** 時（3 IPPON 達成），總分強制設為 **50 分**，觀眾端顯示「FULL IPPON」，該選手勝出，對方分數設為 0 分。

- **+傷停(MEDIC)**：該方選手傷停，觸發傷停計時；詳見傷停規格如對打的傷停規格，時間限上120秒。
- VR(回放)：點選後可觀看該場次的比賽回放，供裁判確認犯規情況；回放結束後自動返回裁判操作畫面。觀眾端畫面則持續顯示比賽狀態，顯示目前在「回放影片」。在按一次就關閉／結束「回放影片」的區塊，回到正常的比賽畫面。
- 最終勝負以裁判判決按鈕確認為準

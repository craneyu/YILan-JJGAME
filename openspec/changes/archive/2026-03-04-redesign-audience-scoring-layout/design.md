## Context

目前的觀眾畫面（創意演武）在顯示分數時，技術分、表演分與大總分採垂直排列，導致資訊密度較低且空間利用不佳。觀眾在大螢幕上關注的是各項細分與最終得分，大總分（技術 + 表演）的顯示顯得冗餘。此外，計時器與隊伍資訊的佈局也需要更緊湊的規劃，以適應不同尺寸的顯示設備。

## Goals / Non-Goals

**Goals:**
- 重新規劃 `CreativeAudienceComponent` 的佈局，使其更現代化、緊湊。
- 將技術分與表演分整合至同一行顯示。
- 隱藏或移除大總分顯示。
- 優化計時器與隊伍基本資訊的排版層次。

**Non-Goals:**
- 不涉及資料庫 Schema 或後端邏輯的變動。
- 不影響雙人演武（Duo）的觀眾畫面。

## Decisions

### 觀眾畫面佈局重新規劃
**決策**：採用水平分欄或更緊湊的垂直堆疊方式，將隊伍資訊置於上方，計分與計時區塊置於下方中心。
**理由**：提煉核心資訊，減少空白區域，使觀眾能一眼看清關鍵賽況。

### 分數整合單行顯示
**決策**：將技術分（Technical）與表演分（Artistic）以左右並排的方式呈現於同一列中。
**理由**：減少垂直空間佔用，並讓兩項核心分數在視覺上具有對等的重要性。

### 移除大總分顯示
**決策**：在 `CreativeAudienceComponent` 的 HTML 中移除大總分的顯示區塊。
**理由**：大總分（技術 + 表演）僅為中間過程，觀眾主要關注細分與最終扣除違例後的得分。

## Risks / Trade-offs

- [UI 顯示不全] → 在較小螢幕或低解析度投影機上，水平並排可能會導致擠壓。
  - **對策**：使用 Tailwind 的響應式設計（sm/md/lg 斷點）來調整排版。
- [觀眾習慣變更] → 原本習慣看大總分的觀眾可能需要適應。
  - **對策**：確保「最終得分」的視覺權重最高，使其成為視覺焦點。
